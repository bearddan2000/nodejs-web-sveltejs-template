# nodejs-web-sveltejs-template

## Description
A template for sveltejs projects.

## Tech stack
- sveltejs
- nginx

## Docker stack
- node:latest
- nginx:latest

## To run
`sudo ./install.sh -u`
- [Avaible here](http://localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
